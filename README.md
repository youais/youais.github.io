# Yiwei Chai

Hello! I’m Yiwei Chai, a PhD student at Johns Hopkins University. I study exoplanets and their systems using a variety of observational methods.

My current projects involve the atmospheric characterisation of a multiple sub-Neptune system with JWST, and using simulations to predict the science yield for transiting exoplanet atmospheres with the upcoming Nancy Grace Roman Space Telescope.

Previously, I worked with Christine Chen to investigate the dynamics and spectral composition of a debris disk—brown dwarf system using direct imaging techniques.
